# algorithms-lib
Collection of widely used algorithms in competitive programming contests
Algorithms included are:

## Graph Algorithms:

  * SSSP algorithms: 
    * Dijkstra.

  * APSP Algorithms:
    * Floyd Warshall.
    
  * MST Algorithms:
    * Kruskal.

## Mathematics:
  * Grey Code.
  * Fibonacci O(log n).
  * Binomial Coefficient.
  
## Data Structures:
  * Sparse Table.
  * Disjoint Set (Union-Find).
